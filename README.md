# Image Gallery

A **responsive and interactive image gallery** built with **HTML, CSS, and JavaScript**.  
Browse images by category, search by title, and toggle between light and dark themes.

---

## Features

- Filter images by categories: Nature, Travel, Food, People, Architecture
- Real-time search by image title
- Responsive design for desktop and mobile
- Dark/Light mode toggle
- Hover overlay showing image title and category
